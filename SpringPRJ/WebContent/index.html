Who We Are

OSAM 2021
2021 군장병 공개SW 온라인 해커톤

SeaSky팀은 OSAM 2021,
군장병 공개SW 온라인 해커톤을 통해 빌딩된 팀입니다.

훈련소에서 훈련병들에게 유일하게 힘이되는 것이 "인터넷 편지"입니다.
저희 SeaSky팀은 기존의 인터넷 편지 어플리케이션인 "더캠프"와 "공군 기훈단"을 Legacy System으로 정의하고
새로운 기능을 추가하고 기존 기능을 개선하고자 합니다.
보다 안전하고 편한 인터넷 편지 발송 System을 통해
훈련병들에게 힘을 주고자 이번 프로젝트를 기획했습니다.

Main Function

주요 기능

모바일/PC 반응형 웹

모바일, 테블릿, PC 모두 환경에 상관없이 100% 호환가능한 반응형 웹이예요.

수료후에도 언제든지 열람할 수 있는 인터넷 편지

발송된 모든 메시지는 DB에 저장되어 수료후에도 언제든지 다시 꺼내 볼 수 있어요.

URL 공유로 손쉬운 인터넷편지 작성

번거로운 작성방법을 공유할 필요없이 URL하나만 SNS에 공유하세요.

개인정보를 알려줄 필요없는 편지 작성페이지

훈련병의 개인정보(연락처, 주민번호 앞자리 등) 개인정보 공유없이도 바로 작성할 수 있어요.

익명 편지 발송기능

익명으로 메시지 발송 가능하며, 편지 발송 시 별도의 가입이 필요없어요.

발송 실패시 백업 기능

발송을 실패하거나 훈련소에서 유실되더라도 걱정마세요. 안전하게 보관돼요.

기존의 인터넷 편지는..
입대전 친구나 가족들에게 자신의 SNS 계정을 알려주고
훈련병 정보가 나오면 SNS에 업로드를 부탁해야합니다.

넥스트레터는 입대전에 단 한 번만, URL을 SNS에 업로드 해두면됩니다.

SNS에 내 개인정보를 업로드하기엔 불쾌한데..

기존의 인터넷 편지는..
SNS에 연락처, 주민번호앞자리 등을 민감 개인정보를 게시해야합니다.

넥스트레터는 개인정보 입력없이 URL에 접속해 바로 편지를 작성하면됩니다.

육군 더캠프 해군 교육사 공군 기훈단 넥스트 레터

개인정보 보호

발송 실패 시, 백업

육군은 어플도 있는데, 해군/공군은 모바일에서 PC 버전으로 보는 건 불편해요.

기존의 인터넷 편지는..
육군은 어플리케이션 다운로드와 회원가입후 인터넷편지 작성가능하며
해군/공군은 모바일 화면이 없어 불편합니다.

넥스트레터는 모바일/PC 반응형 웹으로 디바이스에 관계없이,
별도의 회원가입없이 즉시 편지작성이 가능합니다.

인편 작성하다가 자꾸 사라져요.

기존의 인터넷 편지는..
작성하다가 페이지 오류로 초기화되면 다시 작성해야합니다.

넥스트레터는 발송에 실패해도 안전히 복구됩니다.

훈련소 끝나고 나면 인터넷 편지를 다시 꺼내 볼 수 없어요.

기존의 인터넷 편지는..
훈련소 수료후에 편지를 다시 꺼내 볼 수 없으며,
프린트된 종이는 분실/파손의 위험이 있습니다.

넥스트레터는 수료후에도 어디서든지 언제든지 수령했던 인터넷 편지를 확인 할 수 있습니다.

30초면 나만을 위한 인터넷 편지 작성 페이지가 완성됩니다.

회원가입하세요. 입대 정보가 나온뒤 개인 페이지에서 재수정 할 수 있습니다!

로그인하세요. 공유하기 버튼을 눌러 링크를 복사하세요.

개인 SNS에 업로드하고 친구들, 가족들에게 링크를 알려주세요.

인터넷 편지는 이제, 넥스트 레터에게 맡겨 주세요!

자주하는 질문

입대정보가 나오기 전까지 기다리지 마세요.
입대정보가 없어도 가입할 수 있습니다.
입대정보가 나오면 나중에 개인정보 수정페이지에서 수정하세요.
나만의 인터넷편지 페이지를 미리 꾸며보세요.

해군의 "해군 교육사령부"의 인터넷 편지는 본인 확인 인증으로 인해서 넥스트 레터에서 발송이 불가능합니다.
해군과 논의 중에 있으며 기다려주시면 빠른 시일대에 출시하겠습니다.

편지라는 본연의 의미를 더하기위해서 한번 발송된 편지는 삭제할 수 없는 것을 원칙으로 하고 있습니다.

편지를 발송할 때 입력한 비밀번호는 보안을 위해서 암호화하고 있습니다.
따라서 저희 넥스트레터도 조회 할 수가 없습니다.

넥스트레터는 OSAM2021의 Seasky팀 프로젝트로 오픈소스를 지향하고 있습니다.
넥스트레터의 모든 기능을 JAVA 라이브러리로 제공하고 있으니 MIT License에 따라 자유롭게 사용하셔도됩니다.
